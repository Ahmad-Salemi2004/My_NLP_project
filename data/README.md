# Data Directory

## Dataset Information
This project uses the DialogSum dataset from Hugging Face Datasets.

## Dataset Details
- **Name**: DialogSum
- **Description**: A large-scale dialogue summarization dataset
- **Samples**: 13,460 (train: 12,460, validation: 500, test: 1,500)
- **Format**: Dialogue conversations with human-written summaries

## Access
The dataset is automatically downloaded from Hugging Face when running the training code.

## Fields
Each sample contains:
- `id`: Unique identifier
- `dialogue`: Conversation text
- `summary`: Human-written summary
- `topic`: Conversation topic
